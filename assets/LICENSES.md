# Asset Policy

## 현재 상태

- 기본 실행은 코드 생성 플레이스홀더(도형/색상) 기반입니다.
- 상용/원작 Kirby 자산은 저장소에 포함하지 않습니다.

## 팬에셋 사용 정책 (학습 전용)

- 팬에셋/개인 추출 에셋은 **비공개 개인 학습용**으로만 허용합니다.
- 공개 배포(깃허브 퍼블릭, 실행파일 배포, 영상/상업 활용)는 금지합니다.
- 공개가 필요한 경우 반드시 CC0/퍼미시브 대체 자산으로 교체합니다.

## 공개 가능한 대체 자산 후보

1. Kenney Asset Packs (CC0)
- URL: https://kenney.nl/assets
- License: CC0 1.0

2. OpenGameArt CC0 Packs
- URL: https://opengameart.org/
- License: CC0 표기된 팩만 허용

3. itch.io CC0 Asset Packs
- URL: https://itch.io/game-assets/free/tag-cc0
- License: CC0 표기된 팩만 허용

## 기록 규칙

- 외부 자산 추가 시 아래를 이 파일에 반드시 남깁니다.
- 자산명, 원본 URL, 라이선스, 저작권자/크레딧 문구, 사용 범위(비공개/공개)
